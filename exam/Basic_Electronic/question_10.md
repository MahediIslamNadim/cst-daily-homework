# প্রশ্ন–১০: α (Alpha), β (Beta) ও γ (Gamma)-এর সম্পর্ক স্থাপন কর। (৮ নম্বর)

## উত্তর

ট্রানজিস্টরের তিনটি Current Gain হলো—

* **α (Alpha)** = Common Base (CB) Current Gain
* **β (Beta)** = Common Emitter (CE) Current Gain
* **γ (Gamma)** = Common Collector (CC) Current Gain

---

## সংজ্ঞা

[
\alpha=\frac{I_C}{I_E}
]

[
\beta=\frac{I_C}{I_B}
]

[
\gamma=\frac{I_E}{I_B}
]

এবং,

[
I_E=I_C+I_B
]

---

## সম্পর্ক স্থাপন

যেহেতু,

[
I_E=I_C+I_B
]

উভয় পাশে (I_B) দ্বারা ভাগ করলে,

[
\frac{I_E}{I_B}=\frac{I_C}{I_B}+\frac{I_B}{I_B}
]

অর্থাৎ,

[
\boxed{\gamma=\beta+1}
]

আবার,

[
\alpha=\frac{I_C}{I_E}
]

এবং (I_E=I_C+I_B) বসালে,

[
\alpha=\frac{\beta}{\beta+1}
]

অর্থাৎ,

[
\boxed{\alpha=\frac{\beta}{\beta+1}}
]

এবং,

[
\boxed{\beta=\frac{\alpha}{1-\alpha}}
]

---

## গুরুত্বপূর্ণ সম্পর্ক

[
\boxed{\gamma=\beta+1}
]

[
\boxed{\alpha=\frac{\beta}{\beta+1}}
]

[
\boxed{\beta=\frac{\alpha}{1-\alpha}}
]

---

## উপসংহার

সুতরাং, ট্রানজিস্টরের **α, β ও γ**-এর মধ্যে সম্পর্ক হলো—

* **γ = β + 1**
* **α = β / (β + 1)**
* **β = α / (1 − α)**

### পরীক্ষার জন্য মনে রাখো ✍️

* **α = IC / IE**
* **β = IC / IB**
* **γ = IE / IB**
* **γ = β + 1** ⭐ (সবচেয়ে গুরুত্বপূর্ণ)
