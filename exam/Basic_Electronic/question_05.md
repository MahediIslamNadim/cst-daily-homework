# প্রশ্ন–৫: চিত্রসহ Forward Bias ও Reverse Bias-এর কার্যপ্রণালী বর্ণনা কর। (৮ নম্বর)

## উত্তর:

### Forward Bias

**চিত্র:**

```text
(+)
 │
 P ───|>|─── N
 │
(−)
```

### কার্যপ্রণালী

* P-প্রান্তে **Positive (+)** এবং N-প্রান্তে **Negative (−)** সংযোগ করা হয়।
* Depletion Layer পাতলা হয়।
* ডায়োড সহজে বিদ্যুৎ পরিবহন করে।
* কারেন্ট বেশি প্রবাহিত হয়।

---

### Reverse Bias

**চিত্র:**

```text
(−)
 │
 P ───|>|─── N
 │
(+)
```

### কার্যপ্রণালী

* P-প্রান্তে **Negative (−)** এবং N-প্রান্তে **Positive (+)** সংযোগ করা হয়।
* Depletion Layer মোটা হয়।
* ডায়োড বিদ্যুৎ পরিবহন করে না।
* শুধুমাত্র খুব সামান্য Leakage Current প্রবাহিত হয়।

---

## Forward Bias ও Reverse Bias-এর পার্থক্য

| Forward Bias              | Reverse Bias                   |
| ------------------------- | ------------------------------ |
| P → (+), N → (−)          | P → (−), N → (+)               |
| Depletion Layer পাতলা     | Depletion Layer মোটা           |
| কারেন্ট বেশি প্রবাহিত হয় | কারেন্ট প্রায় প্রবাহিত হয় না |

### উপসংহার

* **Forward Bias → ডায়োড ON (কারেন্ট প্রবাহিত হয়)।**
* **Reverse Bias → ডায়োড OFF (কারেন্ট প্রায় প্রবাহিত হয় না)।**
