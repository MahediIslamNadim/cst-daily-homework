# প্রশ্ন–৬: Block Diagram-এর সাহায্যে D.C. Power Supply ব্যাখ্যা কর। (৮ নম্বর)

## উত্তর:

**D.C. Power Supply** হলো এমন একটি সার্কিট যা **AC (Alternating Current)**-কে **DC (Direct Current)**-এ রূপান্তর করে।

### Block Diagram

```text
 AC Supply
     │
     ▼
Transformer
     │
     ▼
Rectifier
     │
     ▼
Filter
     │
     ▼
Regulator
     │
     ▼
 DC Output
```

---

## প্রতিটি অংশের কাজ

### ১. Transformer

* AC ভোল্টেজকে প্রয়োজন অনুযায়ী কমায় বা বাড়ায়।

### ২. Rectifier

* AC-কে Pulsating DC-তে রূপান্তর করে।

### ৩. Filter

* Pulsating DC-এর Ripple দূর করে মসৃণ DC তৈরি করে।

### ৪. Regulator

* আউটপুট ভোল্টেজকে স্থির (Constant) রাখে।

---

## উপসংহার

সুতরাং, **Transformer → Rectifier → Filter → Regulator** এর মাধ্যমে **AC বিদ্যুৎকে স্থিতিশীল DC বিদ্যুতে** রূপান্তর করা হয়।

### মনে রাখার ট্রিক 📝

**T → R → F → R**

* **T** = Transformer
* **R** = Rectifier
* **F** = Filter
* **R** = Regulator
