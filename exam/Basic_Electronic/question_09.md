# প্রশ্ন–৯: Transistor-এর সাহায্যে প্রমাণ কর যে, (I_E = I_C + I_B)। (৮ নম্বর)

## উত্তর

**Transistor** একটি **তিন টার্মিনালবিশিষ্ট Semiconductor Device**। এর তিনটি টার্মিনাল হলো—

* **Emitter (E)**
* **Base (B)**
* **Collector (C)**

---

## চিত্র

```text
              C
              ↑ IC
              │
              │
          B   │
        → IB  │
              │
              E
              ↑ IE
```

---

## কার্যপ্রণালী

* **Emitter Current (IE)** হলো ট্রানজিস্টরের মোট কারেন্ট।
* এই কারেন্টের একটি ছোট অংশ **Base (IB)** দিয়ে প্রবাহিত হয়।
* বাকি বড় অংশ **Collector (IC)** দিয়ে প্রবাহিত হয়।

অতএব,

**Emitter Current = Collector Current + Base Current**

অর্থাৎ,

[
\boxed{I_E = I_C + I_B}
]

---

## উপসংহার

সুতরাং, ট্রানজিস্টরের **Emitter Current (IE)** সর্বদা **Collector Current (IC)** এবং **Base Current (IB)**-এর যোগফলের সমান।

[
\boxed{I_E = I_C + I_B}
]

### পরীক্ষার জন্য মনে রাখো ✍️

* **IE = মোট কারেন্ট**
* **IC = Collector কারেন্ট**
* **IB = Base কারেন্ট**
* **সূত্র:** **(\boxed{I_E = I_C + I_B})**
